# Libraries Used

```
binascii --> used to convert binary to ASCII

urllib --> Urllib package is the URL handling module for python. It is used to fetch URLs (Uniform Resource Locators). It uses the urlopen function and is able to fetch URLs using a variety of different protocols.

datetime --> for manipulating date and time

rdflib --> Library to work with RDF data.  This library contains parsers/serializers for almost all of the known RDF serializations.

sqlalchemy --> object-relational mapper for the Python programming language

```
